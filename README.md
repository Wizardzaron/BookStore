# Get Started

Create an environment
```shell
# macOS/Linux
python3 -m venv .venv

# Windows
py -3 -m venv .venv

```

Activate the environment
```shell
# macOS/Linux
. .venv/bin/activate

# Windows
.venv\Scripts\activate

```

Install the required packages in activated environment
```shell
# activate environment
pip install -r requirements.txt
```
